# Project Domain Knowledge

Store project-specific business references here: terminology, policies, process
descriptions, data definitions, and other material your Agents should understand.

This directory is created per project and is never installed globally. Add and
maintain files manually as the domain evolves.

Optional paid skills may process these files into a structured `CONTEXT.md`, but
no paid skill content is included in the free framework.

Recommended formats: Markdown (`.md`) and plain text (`.txt`). Never store secrets.
